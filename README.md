## General info
  My version of Tic-Tac-Toe in terminal. Try it out! AI is using minimax algorithm 
  with Alpha-beta pruning.

## Setup
```
$ install Python 3
$ py ttt_terminal.py
$ follow the instructions and use number keys to play
```
